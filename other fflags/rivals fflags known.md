``` json
{
"DFIntMaxAltitudePDStickHipHeightPercent": "-24"
}
```

##### Very controllable bounce, only works with negative values, 0 allows you to hover

``` json
    {
        "DFIntMaxAltitudePDStickHipHeightPercent": "-200"
    }
```
